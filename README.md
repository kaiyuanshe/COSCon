## 关于中国开源年会 (COSCon)

中国开源年会（China Open Source Conference，简称 COSCon）是由[开源社](https://kaiyuanshe.cn)于2015年发起并主办的中国年度开源盛会。

### 会议概况

- **全称**: 中国开源年会 (China Open Source Conference)
- **主办方**: 开源社 (KAIYUANSHE)
- **创办年份**: 2015年
- **举办周期**: 每年一届
- **会议性质**: 非营利性开源技术社区大会

### 最新会议: COSCon'25

**第九届 COSCon'25** 于 2025年12月6-7日 在北京成功举办，恰逢开源社成立十周年。

- **时间**: 2024年12月6-7日（周六-周日）
- **地点**: 北京·丽亭华苑酒店
- **主题**: "Open Source, Open Intellience. 众智开源"
- **规模**: 主论坛 + 多个平行专题论坛

### 会议特色

- **主论坛**: 行业领袖主题演讲、COSCon 十周年故事分享
- **专题论坛**: 涵盖开源教育、云原生、Rust、Web3.0、汽车智能化等多个技术领域
- **社区活动**: 开源集市、互动展区、社区交流
- **联合出品**: 与各技术社区深度合作，确保内容专业性和前瞻性

## 开源社 (KAIYUANSHE)

开源社是由中国开源爱好者组成的开源社区联盟，致力于：

- 推广开源文化和价值观
- 连接中外开源社区
- 促进中国开源生态发展
- 为开源爱好者提供交流平台
